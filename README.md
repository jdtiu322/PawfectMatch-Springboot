# Spring Boot PawfectMatch Project

This is a basic Spring Boot project serving as backend for the web application, Pawfect Match

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Java Development Kit (JDK) installed on your local machine.
- Maven installed on your local machine.

## Getting Started

To get a local copy up and running follow these simple steps.

1. Clone the repository:

    ```bash
    git clone https://github.com/your-username/your-project.git
    ```

2. Navigate into the project directory:

    ```bash
    cd your-project
    ```

3. Build the project using Maven:

    ```bash
    mvn clean install
    ```

## Running the Application

To run the Spring Boot application locally, follow these steps:

1. Navigate into the project directory if you're not already there:

    ```bash
    cd your-project
    ```

2. Run the application using Maven:

    ```bash
    mvn spring-boot:run
    ```

3. Once the application has started, open a web browser and navigate to:

    ```http
    http://localhost:8080
    ```

## Additional Information

- You can modify the application properties in `src/main/resources/application.properties` to customize the application settings.
- For more advanced configurations and features, refer to the [Spring Boot documentation](https://docs.spring.io/spring-boot/docs/current/reference/html/).
